# Keras Example

This project uses Git LFS.
Please install Git LFS at first to pull parameter files (\*.h5).

Install Git LFS

```
$ brew install git-lfs
```

Then, clone this

```
$ git lfs clone https://github.com/deer-dslab/keras-example.git
```
